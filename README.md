# Little CKB Tools

A monorepo of tools for CKB (Nervos Network) development.

## Packages

- [script-analysis](./packages/script-analysis) - A tool to check cells and capacity associated with configurable scripts on CKB Mainnet.

## Getting Started

```bash
pnpm install
pnpm build
```

Each package has its own README with specific usage instructions.
